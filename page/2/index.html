GrowingIO 接入预研

GrowingIO 接入预研

GrowingIO 数据采集技术实现

  1. 收费标准是按app端日活量和网站日访问用户量加在一起进行统计收费。

  2. Android端SDK简洁明了，接入工作量适中，可在短期内实现接入。

  3. 提供app通用信息数据采集，另提供自定义维度数据，用于自定义用户数据的采集

  4. 可以采集App页面数据，用户输入信息数据，但sdk当前无法自动获取GPS信息.

  5. 支持自定义行为数据接收服务器，可优先将数据采集到本地服务器做处理后再转发到 GrowingIO 服务器。

  6. 引入GrowingIO，apk包体积无明显增加，0.2M左右。

  6. 非运算符。~ // ~false = true ~false = true;
