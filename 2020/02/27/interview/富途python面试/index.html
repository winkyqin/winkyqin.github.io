针对算法题的随机生成地雷，
如果要求生成的地雷数比较多，怎么才能避免做重复的随机？
我回答的是用元组单独存储坐标，(x,y)，每次从这个元组列表中随机挑选出一个点来作为地雷，这样可以避免无效的随机。
