目录相关命令

1.1 查看目录内容

  • 自动补全 tab
  • 使用过得命令切换 上/下键
  • ctrl + c

1.2. ls命令说明

  • 显示所有文件（包括隐藏文件以 . 开头的文件）
    ls -a

  • cd . 当前目录
    cd .. 进入上层目录

1.3. ls 常用选项

  • -a 显示指定目录下的所有子目录和文件，包括隐藏文件
    ls -a

  • -l 以列表方式显示文件的信息
    ls -l
    ls -l -h
    ls -lh/-hl

  • -h 配合-l以人性化的方式显示文件大小

##1.4 ls 通配符的使用

  • “*” 代表任意个数的字符
  • “?” 代表任意一个字符、
  • “[]” 字符组
  • “[123]” 字符组
  • “[1-3]” 字符组

2. 切换目录

2.1 cd

  • cd 切换到当前用户的主目录（/home/用户名录）
  • cd ~ 切换到当前用户的主目录（/home/用户名录）
  • cd . 保持当前目录不变
  • cd .. 切换到上级目录
  • cd - 在最近的两次工作目录之间来回切换
