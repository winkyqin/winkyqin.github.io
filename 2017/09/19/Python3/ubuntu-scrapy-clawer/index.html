1.3.2 mongodb for ubuntu install

sudo apt-get install mongodb

(install no response input before the line above :
  
    sudo apt-get update
)

    # check the mongodb is start or not  
    gprep mongo -l
    
    # where the directory is the mongo location 
    locate mongo
    
    # mongodb uninstall 
    sudo apt-get --purge remove mongodb mongodb-clients mongodb-server

  • mongodb start and stop

     # start
 sudo systemctl start mongodb       
 # stop 
 sudo systemctl stop mongodb

  • mongodb install error

    • Failed to start mongod.service: Unit not found

      …

    • Failed global initialization: BadValue Invalid or no user locale set. Please ensure LANG and/or LC_* environment variables are set correctly.

  • （全球初始化失败:BadValue无效或没有用户语言环境设置。请确保LANG and/or LC_ 正确设置环境变量）*

    解决方法

  • 方法一： 手动设置变量，只是临时有用，重启后就没用了，需要在环境变量配置文件添加一行。

     sudo vim /etc/profile
 export LC_ALL=C

  • 方法二： 手动更改源环境

     sudo vim /etc/default/locale
 LANG="en_US.UTF-8"
 LANGUAGE="en_US:en"
 LC_ALL="en_US.UTF-8"
