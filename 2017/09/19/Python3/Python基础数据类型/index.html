Python的字符串

最新的Python3版本中，字符串是以Unicode编码的，PYthon的字符串支持多语言。

Python的字符串类型是str，在内存中以Unicode表示，一个字符对应若干个字节。
Python在网络传输和保存到磁盘上时，需要把str变为以字节为单位的bytes。
Python对bytes类型的数据用带b前缀的单引号或者双引号表示

% 运算符用于格式化字符串的。 %s表示用字符串替换， %d表示用整数替换
